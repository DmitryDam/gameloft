# Gameloft Frontend Test

This project is a demo online store showcasing Gameloft products. Built with **React, TypeScript, and Tailwind CSS**.

## Features

- Responsive product carousel
- Shopping cart with quantity and 10% discount for quantities > 5
- Fully styled with Tailwind CSS
- TypeScript for type safety

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite

## Run Locally

```bash
npm install
npm run dev
